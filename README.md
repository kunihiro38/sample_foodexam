# Title

食品業界資格支援のフードイグザム


# Description

食品業界資格支援のfood-exam.jpは、食品業界の資格合格を目指す人を応援する学習サイトです。ユーザー登録をすると以下の資格のweb問題集が利用できるようになります。

* 食品表示検定・初級
* 食品表示検定・中級
